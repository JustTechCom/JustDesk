📚 JustDesk Documentation

JustDesk is an open-source solution designed to make remote desktop connections easy and secure.
This documentation provides everything you need to set up, run, and troubleshoot the project.

🛠 About the Project

JustDesk includes the following core features:

  • 🖥 Start remote desktop sessions
  • 🔐 Secure connections (with encryption support)
  • 🌐 Web-based user interface
  • 📊 Session management and logging
